Jaehrlich.ca - Portfolio website

Active: 2020 - Now

Languages used: AMP HTML, CSS



Building an ultra fast, moble-first AMP website to display my portfolio and technical blog.

With the majority of website visitors coming from moble devices, and load times turning away visitors, web developers can use AMP HTML to design for fast results. This site shows how to create an extremely light weight and responsive site with minimal javascript. Developed using Visual Studio Code and GitHub to push changes to the live site automatically, for rapid development. Modern templating solutions, like bootstrap, are not supported in an AMP HTML site, so basic HTML and CSS is used to create a simple and effective style.
