How to Create an AMP Website


AMP is fast to load and fast to create

AMP websites are great for serving up clean website content, really fast. AMP can both live on it's own, or help desktop-first sites live a second life as a mobile-first website.

The biggest optimization in speed, compared to desktop-first sites, comes from restricting our design desisions to the simple frameworks of HTML, CSS and JavaScript. With that in mind, it's also fast to setup.

Follow these 6 steps to get up and running.



Create a regular HTML file

Start by creating a regular HTML file and add the word 'amp', or the symbol '⚡', in the html tag.

AMP uses the <html ⚡ lang="en"> markup to label an HTML page as an AMP page. Then, add the usual <head> and <body> sections.
